# Frontend Services

This directory contains API communication services and business logic.

## Services

- **API Service**: HTTP client for backend communication
- **Recommendation Service**: Handles recommendation requests
- **Chat Service**: Manages chat API interactions

## Naming Convention

- Use camelCase for service files (e.g., `apiService.ts`)
- Co-locate tests with services using `.test.ts` suffix
- Services should handle error handling and data transformation
