# Frontend Pages

This directory contains top-level page components for the application.

## Pages

- **Home Page**: Budget input form and recommendation display
- **About Page**: Platform information and value proposition
- **AI Chat Page**: Interactive chat interface with AI assistant

## Naming Convention

- Use PascalCase for page files (e.g., `HomePage.tsx`, `AboutPage.tsx`)
- Co-locate tests with pages using `.test.tsx` suffix
- Pages should compose smaller components from the components directory
